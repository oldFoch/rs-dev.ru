Портфолио

Приложение для салона красоты

Описание: программа для автоматизации деятельности маникюрного салона. Позволила клиенту не терять заявки, видеть запись на сеансы онлайн, устанавливать мотивационные и иные стимулирующие выплаты с учетом нагруженности мастеров маникюра. Помимо этого, собственник имеет возможность со смартфона контролировать деятельность менеджеров, оптимизирован состав администраторов в сети салонов.

Также в приложение встроена система лояльности для постоянных посетителей, с возможностью установления личных скидок для "любимых клиентов".

В дополнение к приложению разработан сайт с возможностью онлайн записи к конкретному мастеру.

Возможности:

  1. онлайн запись;
  2. контроль заявок из директа, мессенджеров, сайта;
  3. контроль деятельности администраторов и менеджеров;
  4. бонусная система и система лояльности;
  5. система бонусов и поощрений для отличившихся сотрудников;
  6. статистика посетителей (день, неделя, месяц, квартал, год);
  7. статистика рекламных компаний с дашбордом;
  8. сайт с интеграцией онлайн записи и подгрузкой Инстаграмм.

Ритуальные услуги

Описание: программа для автоматизации салона по оказанию ритуальных услуг. Позволила клиенту учитывать заявки полученные от агентов, соответственно устанавливать мотивационные и иные стимулирующие выплаты с учетом приведенных клиентов. Со смартфона контролировать деятельность салона. Интеграция системы в 1С "Розница".

В приложение встроена система лояльности для постоянных посетителей (ритуальных посредников).

Возможности:

  1. контроль деятельности агентов;
  2. бонусная система и система лояльности;
  3. система бонусов и поощрений для отличившихся агентов;
  4. статистика продаж (день, неделя, месяц, квартал, год).

Медицинская клиника

Описание: программа для автоматизации деятельности наркологической клиники. Позволила клиенту не терять заявки, видеть запись на сеансы онлайн, устанавливать мотивационные и иные стимулирующие выплаты с учетом нагруженности медицинского персонала. Помимо этого, собственник имеет возможность со смартфона контролировать деятельность менеджеров, видит результаты рекламных компаний.

Дополнительно разработана система контроля за выездной "скорой помощью", осуществляется контроль оказанных услуг и использованных препаратов. Система ограничена внутри клиники, ведутся электронные карты клиентов.

Возможности:

  1. контроль заявок из директа, мессенджеров, сайта, офлайн рекламы;
  2. контроль деятельности администраторов и менеджеров;
  3. система бонусов и поощрений для отличившихся сотрудников;
  4. статистика посетителей (день, неделя, месяц, квартал, год);
  5. статистика рекламных компаний с дашбордом.

Риэлторское агентство

Описание: программа для автоматизации деятельности агентства по аренде квартир. Позволила клиенту не терять заявки, видеть запись онлайн, устанавливать мотивационные и иные стимулирующие выплаты с учетом нагруженности специалистов. Помимо этого, собственник имеет возможность со смартфона контролировать деятельность менеджеров, видит результаты рекламных компаний.

Интеллектуальный подбор недвижимости в зависимости от профиля клиента, автоматическая отправка в приоритетные мессенджеры карточек квартир в формате .pdf. Онлайн доступ с телефона для риэлторов позволил заказчику перевести более 80 процентов сотрудников на удаленную работу.

Дополнительно разработана система автоматического обзвона собственников помещений с целью уточнения наличия свободных квартир.
