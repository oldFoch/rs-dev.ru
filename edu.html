Программа обучения:

  1. Знакомство. Понятия "Веб-приложение", "Архитектура", "BackEnd и FrontEnd".
  2. Изучение HTML. Как браузер строит страницу. Понятие DOM. Изучение тегов и аттрибутов.
  3. Изучение CSS. Понятие CSSOM. Написание таблиц стилей.
  4. Изучение CSS. Адаптивная верстка.
  5. Изучение JS. Базовые навыки программирования.
  6. Изучение JS. jQuery и работа с элементами страницы. Подключение файлов в HTML. Организация файлов в проекте.
  7. Изучение системы контроля версий GIT. Создание аккаунта в GitHub.
  8. Изучение PostgresSQL. Понятие БД, таблицы, поля, запросы.
  9. Изучение PostgresSQL. Написание запросов SELECT, INSERT, UPDATE, DELETE.
  10. Изучение PostgresSQL. Написание сложных запросов: JOIN, WHERE, GROUP BY, ORDER BY, WITH.
  11. Изучение Python. Базовые навыки программирования.
  12. Изучение Python. ОПП.
  13. Изучение Python. Работа с файлами. Разработа парсера сайтов.
  14. Изучение Python. Понятие API. Создание бота Telegram.
  15. Изучение Flask. Понятие конечной точки (endpoint). Изучение протокола HTML.
  16. Изучение Flask. Понятие CROSS-ORIGIN запросов. Понятие ORM, миграции. Создание связи с БД.
  17. Изучение Flask. Простое серверное приложение CRUD.
  18. Изучение VueJS. Понятие реактивности. Основные механики работы.
  19. Изучение VueJS. Понятие компонента. Разработка простого приложения.
  20. Изучение VueJS. Изучение Vuex, vue-router.
  21. Разработка приложения. Объединение BackEnd и FrontEnd.
  22. Разработка приложения. Продумывание архитектуры. Разработка схемы БД.
  23. Разработка приложения. Продумывание дизайна. Понятия UI/UX.
  24. Разработка приложения. Разработка интерфейса.
  25. Разработка приложения. Разработка бизнес - логики.
  26. Дипломный проект. Разработка.
  27. Финиш! Вручение сертификатов.
